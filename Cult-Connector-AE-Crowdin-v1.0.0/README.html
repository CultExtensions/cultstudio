Cult Connector

ScriptUI panel for Adobe After Effects. Send compositions and screenshots to Crowdin, translate with full visual context, and import localized comps—without leaving your timeline.

ZIP install macOS & Windows Crowdin Teams & Enterprise

Requirements

  • Adobe After Effects (recent year build recommended)
  • Crowdin access and a Cult Extensions account (trial available)
  • Network permission for the panel (see After Effects scripting prefs below)

Before first launch — After Effects scripting

  1. Open After Effects → Preferences (macOS) or Edit → Preferences (Windows) → Scripting & Expressions.
  2. Enable Allow Scripts to Write Files and Access Network.
  3. Quit After Effects completely and reopen so the setting applies everywhere.

Most Cult Connector features need network access (Crowdin API, Cult Extensions auth). Without this, the panel may appear blank or fail silently.

Install on macOS

  1. Open After Effects.
  2. Go to File → Scripts → Install Script UI Panel…
  3. Select Cult Connector (AE ↔ Crowdin).jsxbin.
  4. Restart After Effects.
  5. Open Window → look for Cult Connector (AE ↔ Crowdin) (name matches the script).

If you install a new AE version later, you may need to run “Install Script UI Panel…” again for that version.

Install on Windows

  1. Open After Effects.
  2. Go to File → Scripts → Install Script UI Panel…
  3. Select Cult Connector (AE ↔ Crowdin).jsxbin.
  4. Restart After Effects.
  5. Open Window → look for Cult Connector (AE ↔ Crowdin) (name matches the script).

If Windows blocks the install prompt, run After Effects as Administrator once, install the panel, then reopen normally.

Using Cult Connector (quick)

  1. Dock the panel where you like.
  2. Sign in via the browser flow when prompted (stay signed in when the panel expects it).
  3. Connect your Crowdin project (Teams or Enterprise).
  4. Use the Composition workflow to send strings; add markers on tricky motion for exact screenshot frames.
  5. Import translations back when ready—typically one composition per language.
