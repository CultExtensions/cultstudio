Cult Translator

ScriptUI panel for Adobe After Effects. Translate text layers in-place with an AI engine, optional context, and a clean browser sign-in flow.

ZIP install macOS & Windows Works with Cult Studio seats

Before first launch — After Effects scripting

  1. Open After Effects → Preferences (macOS) or Edit → Preferences (Windows) → Scripting & Expressions.
  2. Enable Allow Scripts to Write Files and Access Network.
  3. Quit After Effects completely and reopen so the setting applies everywhere.

Cult Translator needs network access for Cult Studio authentication and AI translation.

Install on macOS

  1. Open After Effects.
  2. Go to File → Scripts → Install Script UI Panel…
  3. Select Cult Translator.jsxbin.
  4. Restart After Effects.
  5. Open Window → Cult Translator (name matches the script).

Install on Windows

  1. Open After Effects.
  2. Go to File → Scripts → Install Script UI Panel…
  3. Select Cult Translator.jsxbin.
  4. Restart After Effects.
  5. Open Window → Cult Translator (name matches the script).

If Windows blocks the install prompt, run After Effects as Administrator once, install the panel, then reopen normally.

How to use (quick)

  1. Open a composition. Select one or more text layers (optional).
  2. Click Sign in in the panel and complete the browser flow.
  3. Add Context (optional): describe the scene/brand/tone so the translation matches your intent.
  4. Choose source/target languages and click Translate.

If you see “Not allowed for this workspace”, ask your admin to add your email under Cult Translator → Users in Cult Studio, then click Refresh in Settings.
